## 使用方式

1. 在 MoviePilot 中添加自定义插件仓库：`https://github.com/lucku88/MoviePilot-Plugins/`
2. 刷新插件仓库并安装需要的插件。
3. 在插件配置页启用插件，并按需同步站点 Cookie。
4. `Vue-农场` 如需验证码识别，再额外配置 OCR 地址。

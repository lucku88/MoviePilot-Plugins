## 插件说明

### 🌱 Vue-农场
- 状态页、配置页和主要操作直接以 [KoWming/Vue-思齐农场 v1.0.1](https://github.com/KoWming/MoviePilot-Plugins/releases/tag/SiqiFram_v1.0.1) 的源码为底稿移植，再接入本仓库原有的稳定自动化功能。
- 收菜继续使用动态调度：识别每块田的真实成熟时间，并在成熟后增加缓冲秒数再运行，不使用固定 4 小时周期。
- 手动种植同样会登记动态收菜计划；成熟前预刷新后会强制注册正式收菜，即使作物临时消失也会保留原计划，到点运行并自动补种空地。
- 立即运行或页面种植后会等待新作物写入田地状态，并按该种子的生长时间保底排期；只要存在成熟田块，就会优先立即收菜，不会被其他远期作物时间跳过。
- 支持自动收菜、优先种植、一键种植、一键出售、逐坑位操作和一行执行历史。
- 默认启用验证码批量收菜并使用 MoviePilot 内置 OCR，也可切换到自建 TRWebOCR 容器。
- OCR 失败后可选用 MoviePilot AI 辅助识别；OCR 和 AI 都失败、或批量后发现漏收时会立即逐坑位补收。整次收菜默认限制为 45 秒，并为逐坑位阶段预留时间。
- 优先种植会兼容带逗号的总收获数字，并信任种子已解锁状态，避免蘑菇等新解锁种子被误判为未解锁。
- 自动偷菜支持同时选择多个作物、每轮访问人数和多个时间段；随机到所选作物时会按网站当天剩余额度尽量连续偷取，额度未用完会继续检查，网站提示次数耗尽后才停止当天任务。
- 偷菜通知按实际成功结果汇总，例如 `🌽玉米×3`，不会再把所有已选择作物都写进通知。
- 自动点赞支持三个固定用户名，未填写的位置会用不重复的随机目标补齐；执行时间使用带可读解释的 Cron 设置，默认 `0 9 * * *`（每天 09:00），错过后当天会自动补跑。
- 配置页的“运行一次”会执行主农场任务，并同步执行已开启的偷菜和点赞；旧版本保存的 JSON 格式多选作物会自动修正显示。
- 站点 Cookie 固定自动同步，无需配置开关；插件启动、保存和请求前都会读取 MoviePilot 站点管理中的最新 Cookie，手动输入仅作同步失败备用。
- 支持手动进入随机农场选择坑位偷菜、随机填充点赞名单和批量点赞；当天无点赞目标时不会反复请求。
- 网络异常单次请求最多自动重试 5 次，连续失败超过 5 轮后停止短间隔补跑，避免夜间刷屏。

### ⚗️ Vue-魔丸

- 支持保存配置、启动初始化和预刷新后的自动补跑，避免重启后错过已到时间的任务。

- 支持保存配置、启动初始化和预刷新后的自动补跑，可优先处理自家展位可收回和可补位场景。

- 支持保存配置、启动初始化和预刷新后的自动补跑，覆盖待收开包、可开包和舞台召回 / 演出场景。

## 使用方式

1. 在 MoviePilot 中添加自定义插件仓库：`https://github.com/lucku88/MoviePilot-Plugins/`
2. 刷新插件仓库并安装需要的插件。
3. 在插件配置页启用插件；站点 Cookie 会自动从 MoviePilot 站点管理中读取。
4. `Vue-农场` 默认不需要 OCR；只有要使用批量收菜时，才打开 `OCR批量收菜` 并填写 OCR API 地址。

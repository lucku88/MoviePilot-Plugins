## 插件说明

### 🌱 Vue-农场
- 收菜、种植、出售、获取执行记录。
- 默认不启用 OCR 批量收菜，成熟后会直接逐坑位快速收菜并复查漏收。
- 如需 OCR 批量收菜，可在配置页打开 `OCR批量收菜` 并填写 OCR API 地址；验证码 3 次失败或复查发现漏收时，会自动切换逐坑位补收。
- 优先种植会兼容带逗号的总收获数字，并信任种子已解锁状态，避免蘑菇等新解锁种子被误判为未解锁。
- 网络异常单次请求最多自动重试 5 次，连续失败超过 5 轮后停止短间隔补跑，避免夜间刷屏。

### ⚗️ Vue-魔丸
- 兑换、搬砖、清沙滩、炼造、获取执行记录。
- 支持保存配置、启动初始化和预刷新后的自动补跑，避免重启后错过已到时间的任务。

### 🧸 Vue-玩偶
- 盲盒、回收、展出、获取执行记录。
- 支持保存配置、启动初始化和预刷新后的自动补跑，可优先处理自家展位可收回和可补位场景。

### 🎭 Vue-表情
- 老虎机、开包、舞台演出、获取执行记录。
- 支持保存配置、启动初始化和预刷新后的自动补跑，覆盖待收开包、可开包和舞台召回 / 演出场景。

### 📊 Vue-面板
- 个人用模块化面板。
- 新增功能卡片开发说明见 `plugins.v2/vuepanel/ADD_CARD_GUIDE.md`。

### 🎞️ 飞牛影视媒体库封面生成
- 生成静态封面并回写飞牛影视。

## 使用方式

1. 在 MoviePilot 中添加自定义插件仓库：`https://github.com/lucku88/MoviePilot-Plugins/`
2. 刷新插件仓库并安装需要的插件。
3. 在插件配置页启用插件，并按需同步站点 Cookie。
4. `Vue-农场` 默认不需要 OCR；只有要使用批量收菜时，才打开 `OCR批量收菜` 并填写 OCR API 地址。
